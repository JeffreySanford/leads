# Review Scripts Documentation

This folder contains scripts for managing ND NAICS leads and probing/searching SAM data.

## Scripts
- `nd_pack.sh`: Packs ND NAICS leads into a CSV file.
- `sam_probe.sh`: Probes SAM for lead information.
- `sam_probe_verbose.sh`: Verbose probe of SAM for lead information.
- `sam_search.sh`: Searches SAM for leads.
- `nd_naics_leads.csv`: Sample CSV data for ND NAICS leads.

## Usage
- Run each script from the command line using bash.
- See individual script comments for usage details.

## Example
```bash
./nd_pack.sh
./sam_probe.sh 1001
./sam_probe_verbose.sh 1001
./sam_search.sh Acme
```
