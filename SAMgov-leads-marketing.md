# SAM.gov Lead Tracking System – Key Features & Target

## Target Audience
- North Dakota IT businesses
- Government contractors
- Procurement teams seeking federal contract opportunities

## Core Features
- Real-time monitoring of SAM.gov API for new contracts
- Database integration for sample and live contract records
- Toggle to show all NAICS-connected records (sample + live), with pagination and record/contract totals
- Status indicators for Backend API, Database, and SAM.gov API
- Material table with filtering, pagination, and columns for date posted, contract value, and title

## Business Value
- Focus on set-aside contracts under $250K for small businesses
- Option to highlight $10K gigs that don’t require full processing
- Fast access to current and historical contract data for strategic decision-making
- User-friendly dashboard for quick filtering and review

## Marketing Angle
- **Find and track federal IT contracts in North Dakota—instantly.**
- **See all NAICS-connected opportunities, including small business set-asides and micro-purchases.**
- **Monitor, filter, and act on new contracts with real-time data and easy-to-use tools.**
