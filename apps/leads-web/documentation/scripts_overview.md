# Scripts Overview

## nd_pack.sh

Packs ND NAICS leads into a CSV file. Copies `nd_naics_leads.csv` to `packed_nd_leads.csv`.

## sam_probe.sh

Probes SAM for lead information. Accepts a lead ID as an argument and outputs sample results.

## sam_probe_verbose.sh

Provides a verbose probe of SAM for lead information. Accepts a lead ID and outputs detailed sample results.

## sam_search.sh

Searches SAM for leads using a search term. Outputs sample search results.

## nd_naics_leads.csv

Sample data file containing ND NAICS leads.
